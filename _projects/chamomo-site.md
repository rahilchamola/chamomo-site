My personal website — the one you're looking at right now.

Built from scratch with Jekyll. No theme, no template. Custom dark theme with a design system that grows with the site. Deployed on Vercel.

## Why From Scratch

Templates optimize for looking good on day one. I'm optimizing for understanding the system deeply enough to evolve it over years. Every layout, component, and style decision is intentional and documented.

## Architecture

The site uses Jekyll's collection system to organize four content types: writing (blog posts), projects, ideas (a digital garden with seed/growing/mature status), and DJ content. Each has its own layout, front matter schema, and listing page.

The CSS architecture uses Sass with a token-based design system — variables for colors, typography, spacing, and breakpoints that make the entire visual language changeable from one file.

## What I'm Learning

Web development fundamentals. Liquid templating. CSS architecture. How to structure content for longevity. How to maintain a site through Claude.
