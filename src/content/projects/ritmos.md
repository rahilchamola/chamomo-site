## What Ritmos Is

Ritmos is a DJ set design tool. You give it a brief — theme, target BPM arc, crowd context, recognition density — and it outputs an ordered setlist with transition rationale at every step. Not a playlist generator. A set architect.

## The Two Kinds of Work

DJing has technical work (BPM matching, key compatibility, energy management) and intuitive work (reading the room, live decisions, crowd energy). These require different attention. Ritmos moves all technical work into preparation so every unit of attention during performance is available for the room.

## The Naming

Ῥυθμός — Greek for the pattern of movement through time. Not "music." The name encodes the thesis: a DJ set is a shape, not a list. A BPM progression is a rhythmic argument.
